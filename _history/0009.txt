# 작업 기록 - GitHub 원격 저장소 설정

## 날짜: 2023년 5월 27일

## 작업 내용
1. GitHub 원격 저장소 연결 문제 해결
   - 기존 'origin' 원격 저장소 확인 (git remote -v)
   - 기존 'origin' 원격 저장소 제거 (git remote remove origin)
   - 새 'origin' 원격 저장소 추가 (git remote add origin https://github.com/younsanghoon/PalmReading.git)
   - 원격 저장소 설정 확인 (git remote -v)
   - 메인 브랜치를 원격 저장소에 푸시 시도 (git push -u origin main)

## 결과
- GitHub 원격 저장소 연결 성공
- 프로젝트를 GitHub 저장소에 푸시 준비 완료
