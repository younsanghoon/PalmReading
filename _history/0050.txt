# 동물상 테스트 결과 표시 문제 추가 수정 2

## 문제 분석
- 동물상 테스트에서 '유형E'로 결과가 표시되는 문제 발생
- 이미지 분석 시 '공룡상(dinosaur)'이 결과로 나왔으나 해당 동물상 정보가 없어 오류 발생
- 사용자가 제공한 스크린샷에서 유형E가 32.8% 확률로 표시됨

## 원인 파악
- `client/src/lib/personality-data.ts`의 `ANIMAL_PERSONALITIES` 객체에 '공룡상' 항목이 누락됨
- '사슴상' 항목도 누락되어 있었음
- 모델은 'dinosaur' 클래스를 인식하지만 해당 한글 동물상 정보가 없어 결과 표시에 문제 발생

## 수정 내용
1. `client/src/lib/personality-data.ts` 파일 수정
   - `ANIMAL_PERSONALITIES` 객체에 '공룡상' 항목 추가
   ```typescript
   '공룡상': {
     traits: ['독특한', '강인한', '대담한', '인내심 있는', '독립적인'],
     description: '공룡상은 독특하고 강인한 성격으로 자신만의 개성이 뚜렷한 타입입니다...',
     personality: '독특하고 강인한 성격으로 자신만의 개성이 뚜렷한 타입입니다.',
     charm: '어려운 상황에서도 굳건히 자신의 길을 걸어가는 강한 의지가 매력적입니다.',
     dating: '신뢰와 존중을 바탕으로 한 관계를 추구하며 파트너에게 안정감을 줍니다.',
     en: { ... }
   }
   ```
   
   - `ANIMAL_PERSONALITIES` 객체에 '사슴상' 항목 추가
   ```typescript
   '사슴상': {
     traits: ['온순한', '우아한', '차분한', '순수한', '섬세한'],
     description: '사슴상은 온순하고 우아한 성격으로 차분하고 섬세한 매력을 가진 타입입니다...',
     personality: '온순하고 우아한 성격으로 차분하고 섬세한 매력을 가진 타입입니다.',
     charm: '맑고 순수한 눈빛으로 주변 사람들에게 편안함을 줍니다.',
     dating: '조용하고 깊은 관계를 추구하며 파트너를 세심하게 배려합니다.',
     en: { ... }
   }
   ```

## 기대 효과
- 모든 동물상 유형에 대한 정보가 제공되어 결과 표시 오류 해결
- '공룡상'과 '사슴상'에 대한 성격 정보가 정상적으로 표시됨
- 사용자가 모든 동물상 유형에 대한 결과를 정확히 확인 가능
