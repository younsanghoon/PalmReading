# 작업 내역: 동물상 테스트 페이지 오류 수정

## 문제 상황
- 동물상 테스트(animal-test) 페이지에서 `Cannot destructure property 'modelURL' of 'undefined' as it is undefined.` 오류 발생
- 브라우저 콘솔에 오류 메시지가 표시되며 테스트 실행이 불가능한 상태

## 조사 과정
1. animal-face-test.tsx 컴포넌트 코드 검토
2. useTeachableMachine.ts 훅 검토
3. 오류 메시지 분석

## 문제 원인
1. `useTeachableMachine` 훅 호출 시 필수 매개변수인 `modelURL`과 `metadataURL`을 전달하지 않음
2. 이전 손금 테스트와 동일한 문제로, 모델 URL이 정의되지 않은 상태에서 훅을 사용하려고 시도

## 수정 내용
1. animal-face-test.tsx
   - `modelURL`과 `metadataURL` 상수 추가 및 `useTeachableMachine` 훅에 전달
   - 오류 처리 로직 개선 (try-catch 추가)
   - 분석 실패 시 사용자에게 피드백 제공

2. use-teachable-machine.ts
   - `predictAnimal` 함수 추가 (기존 `predict` 함수를 래핑)
   - 동물상 테스트 전용 예측 함수 구현

3. 예외 처리 개선
   - 모델 로딩 실패 시 적절한 에러 메시지 표시
   - 예측 결과가 없을 때 사용자 피드백 제공

## 결과
- 동물상 테스트 페이지가 정상적으로 로드되고 실행됨
- 모델 로딩 및 예측 과정에서 오류 발생 시 적절한 피드백 제공
- 이전 손금 테스트와 동일한 패턴으로 문제 해결

## 교훈
- 외부 라이브러리나 커스텀 훅 사용 시 필수 매개변수 확인 필요
- 비동기 작업에 대한 적절한 예외 처리 중요
- 사용자에게 오류 상황을 명확히 전달하는 피드백 제공 필요
