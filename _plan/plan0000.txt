# PalmReading 프로젝트 거시적 계획

## 프로젝트 개요
PalmReading은 AI와 심리학을 기반으로 한 성격 분석 테스트 웹 애플리케이션입니다. 사용자들은 동물상 테스트, MBTI 테스트, 에니어그램 테스트, 손금 테스트 등 다양한 성격 분석 도구를 이용할 수 있습니다. 특히 손금 분석 기능은 AI를 활용하여 사용자의 손금 사진을 분석하고 결과를 제공합니다.

## 최종 목적
1. 프로젝트의 구조와 기능을 완전히 이해하고 분석
2. 애플리케이션을 성공적으로 실행하고 모든 기능이 정상 작동하는지 확인
3. 필요한 경우 코드 개선 및 버그 수정
4. 프로젝트 문서화 및 향후 개발 방향 제안

- 모델 성능 및 정확도 평가

### 3. 애플리케이션 실행 및 테스트
- 개발 환경 설정 및 의존성 설치
- 클라이언트 및 서버 실행

- PWA 기능 검증

### 4. 코드 품질 및 성능 평가
- 코드 구조 및 가독성 평가

- 개선 가능한 영역 식별 및 제안

## 기술 스택
- 프론트엔드: React, TypeScript, TailwindCSS
- 백엔드: Express.js, Node.js
- AI: TensorFlow.js, Teachable Machine
- 상태 관리: React Query
- 라우팅: Wouter
- UI 컴포넌트: Radix UI
- 데이터베이스: Drizzle ORM (NeonDB)
- 배포: PWA (Progressive Web App)

## 주요 기능
1. 동물상 테스트 (AI 얼굴 분석)
2. MBTI 성격 유형 테스트
3. 에니어그램 테스트
4. 손금 분석 테스트 (AI 이미지 분석)
5. 다국어 지원 (한국어, 영어, 일본어, 중국어, 인도네시아어, 태국어, 베트남어)
6. PWA 지원 (오프라인 사용 가능)
7. 결과 공유 기능

## 실행 계획
1. 프로젝트 의존성 설치 (npm install)
2. 개발 모드로 애플리케이션 실행 (npm run dev)
3. 각 테스트 기능 검증
4. 빌드 및 프로덕션 환경 테스트 (npm run build, npm start)
5. 분석 결과 및 개선 사항 문서화
