# PalmReading 프로젝트 분석 - GitHub 배포 준비

## GitHub 원격 저장소 설정

### 원격 저장소 연결 문제 분석
- 초기 시도: `git remote add origin https://github.com/younsanghoon/PalmReading.git`
- 발생 오류: `error: remote origin already exists.`
- 원인: 이미 다른 URL로 'origin' 원격 저장소가 설정되어 있음
- 해결 방법: 기존 원격 저장소 제거 후 새 원격 저장소 추가

### 원격 저장소 설정 과정
1. 기존 원격 저장소 확인
   ```bash
   git remote -v
   # 결과: origin https://github.com/사용자이름/PalmReading.git (fetch)
   ```

2. 기존 원격 저장소 제거
   ```bash
   git remote remove origin
   ```

3. 새 원격 저장소 추가
   ```bash
   git remote add origin https://github.com/younsanghoon/PalmReading.git
   ```

4. 설정 확인
   ```bash
   git remote -v
   # 결과: 
   # origin https://github.com/younsanghoon/PalmReading.git (fetch)
   # origin https://github.com/younsanghoon/PalmReading.git (push)
   ```
